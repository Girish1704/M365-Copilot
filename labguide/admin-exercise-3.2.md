# Exercise 3.2: Administering M365 Copilot

In this exercise, you will explore the services included with the **Microsoft 365 Copilot** license. You will navigate to the **Microsoft 365 Admin Center**, open a user's license details, and review the list of enabled Copilot services.

### Task 1: Review Microsoft 365 Copilot Services

**Microsoft 365 Copilot** includes several services that can be enabled or disabled based on your organization's needs. You can view these services from the **Microsoft 365 Admin Center**.

1. Navigate to the **Microsoft 365 Admin Center** using the URL below and sign in with your admin credentials.

    ```
    https://admin.microsoft.com/
    ```

1. From the left navigation pane, click on **Users (1)** and then select **Active users (2)**.

    ![](../labguide/media/adm-cop-3.1-g1.png)

1. On the **Active users** page, select **ODL_User <inject key="DeploymentID" enableCopy="false"/>**.

    ![](../labguide/media/adm-cop-3.1-g2.png)

1. On the user profile page, click on **Licenses and apps** on the right side to view the license details.

    ![](../labguide/media/adm-cop-3.1-g3.png)

1. Scroll down to the **Apps** section. You will see a list of enabled services tied to the **Microsoft 365 Copilot** license. 
 
    ![](../labguide/media/adm-cop-3.1-g21.png)
   
   Below is a brief overview of each **Microsoft 365 Copilot** service:

    - **Copilot Studio in Copilot for Microsoft 365:** This service allows you to customize and extend Copilot across Microsoft 365 apps. You can create custom workflows, integrate third-party or internal tools, and manage specific actions. It supports no-code and low-code customization.
      
    - **Graph Connectors in Microsoft 365 Copilot:** These connectors allow Copilot to access data from **Microsoft Graph**. This helps Copilot pull information from various Microsoft services, so it can provide more relevant and accurate responses.

    - **Intelligent Search:** This service uses AI to improve search results across Microsoft 365 apps. Instead of matching keywords only, it understands the context of your query and returns more useful results.

    - **Microsoft 365 Copilot for SharePoint:** This service helps you with content creation and management in SharePoint. It can generate and refine content for pages and news posts, suggest metadata, and summarize discussions.

    - **Microsoft 365 Copilot in Microsoft Teams:** In Teams, Copilot helps you summarize conversations, suggest next steps, and assist with meeting notes. It turns discussions into actions so you can stay organized.

    - **Microsoft 365 Copilot in Productivity Apps:** Copilot is built into apps like Word, Excel, and PowerPoint. It can provide suggestions, automate tasks, and generate content. For example, it helps you design slides in PowerPoint or draft documents in Word.
  
    - **Microsoft Copilot with Graph-Grounded Chat:** This service uses Microsoft Graph to give you an intelligent chat experience. Copilot can access your organizational data such as files, meetings, and emails, so its responses are based on real-time, relevant information.

    - **Power Platform Connectors in Microsoft 365 Copilot:** These connectors let Copilot work with the **Power Platform**, which includes Power Apps, Power Automate, Power BI, and Power Virtual Agents. This allows Copilot to access data, run operations, or trigger workflows within these tools.
